# Compiler

Компилятор подмножества языка Java. Проект предназначен для сборки исходного кода и запуска тестов через скрипт автоматической проверки. CMake — стандартный инструмент для конфигурации и сборки проектов, а `run_all_tests.sh` используется для запуска набора тестов [web:7][web:5].

## Требования

Для сборки и запуска проекта нужны:
- `cmake`.
- `make`.
- `bash`.
- Установленная среда для запуска собранного бинарника.

## Сборка

Сборка выполняется из корня проекта следующими командами:

```bash
mkdir build && cd build && cmake .. && make && cd ..
```

После успешной сборки исполняемый файл будет находиться в `build/Compiler`, а запуск производится с указанием входного и выходного файлов:

```bash
./build/Compiler input out
```

## Запуск тестов

Для запуска всех тестов используйте скрипт:

```bash
./run_all_tests.sh
```

Скрипт последовательно запускает весь набор тестов проекта и выводит результаты проверки в консоль [web:5].

## Структура запуска

Типичный рабочий цикл выглядит так:
1. Собрать проект.
2. Запустить компилятор на нужном входном файле.
3. Проверить корректность работы через `run_all_tests.sh`.

## Пример

```bash
mkdir build && cd build && cmake .. && make && cd ..
./build/Compiler mini-java out
./run_all_tests.sh
```
